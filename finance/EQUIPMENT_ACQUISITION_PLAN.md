# Equipment Acquisition Plan

## Goal
Build a reliable starter setup for grass cutting and power washing without overspending before recurring customers exist.

## Required Before Grass Cutting Launch
### Core Lawn Equipment
- [ ] mower
- [ ] string trimmer
- [ ] leaf blower
- [ ] fuel containers
- [ ] extension cords/battery setup if applicable
- [ ] spare blades
- [ ] trimmer line

### Safety Equipment
- [ ] eye protection
- [ ] hearing protection
- [ ] gloves
- [ ] work boots
- [ ] reflective/work shirts
- [ ] first aid kit

### Transport
- [ ] truck or reliable transport vehicle
- [ ] trailer if needed
- [ ] tie-down straps
- [ ] ramps if needed

## Required Before Power Washing Launch
### Power Washing Core Equipment
- [ ] pressure washer
- [ ] high-pressure hose
- [ ] spray tips/nozzles
- [ ] surface cleaner
- [ ] extension wand
- [ ] hose reels if needed
- [ ] water hose
- [ ] chemical sprayer if used

### Power Washing Safety
- [ ] waterproof boots
- [ ] waterproof gloves
- [ ] eye protection
- [ ] hearing protection
- [ ] rain gear / waterproof clothing

## Helpful But Not Required Immediately
- [ ] commercial mower
- [ ] enclosed trailer
- [ ] branded wrap/signage
- [ ] dedicated business phone
- [ ] advanced CRM software
- [ ] commercial pressure washing rig
- [ ] water tank system

## Do Not Overbuy Early
Avoid large equipment purchases before:
- recurring route density exists
- pricing is validated
- demand is proven
- workflow is stable
- transport is reliable

## Priority Rule
Reliable operation matters more than expensive branding during early launch.
