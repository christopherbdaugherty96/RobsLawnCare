# Rebrand Source Model

## Previous Working Identity
Rob's Lawn Care

## New Brand Identity
ShortCut Lawn Care

## Correction
ShortCut is not a separate business inspired by Rob's Lawn Care.

ShortCut is the new name and brand direction for the same lawn care business idea.

## What Carries Forward
The original Rob's Lawn Care concept should carry forward into ShortCut:
- simple local lawn care service
- recurring mowing
- trimming and edging
- cleanup services
- fast quotes
- before/after proof
- neighborhood reputation building
- seasonal upsells

## What The Rebrand Should Improve
ShortCut should improve the original idea with:
- cleaner branding
- sharper name
- better mobile website
- simpler quote flow
- stronger local positioning
- more organized operating system

## Positioning Shift
Old framing:
Rob's Lawn Care as a local lawn care concept.

New framing:
ShortCut as the cleaner, more scalable brand for that same business.

## Important Rule
Future docs should not treat Rob's Lawn Care as a competitor unless specifically analyzing outside market competitors.
Rob's Lawn Care is the prior concept/source identity. ShortCut is the active brand.
