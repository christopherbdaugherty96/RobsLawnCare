# Gallery Structure

## Purpose
Define how ShortCut should organize before/after photos once real work exists.

## Current Phase
Pre-launch planning.

The gallery should not use misleading stock photos or imply completed work that has not happened.

## Gallery Sections
### Grass Cutting
Use for:
- mowing results
- edging results
- cleanup before/after
- overgrowth recovery

### Power Washing
Use only after power washing work is actually completed and safe to show.

Potential categories:
- driveway cleaning
- walkway cleaning
- patio cleaning
- exterior surface cleaning

## Photo Rules
- use real work photos only
- get customer permission when appropriate
- avoid showing private details
- avoid showing license plates, addresses, or personal items when possible
- do not exaggerate results

## Before/After Workflow
For each job when appropriate:
1. take before photo
2. complete service
3. take after photo
4. record service type
5. record date and general area
6. request permission if customer property is identifiable

## Website Rule
The gallery should build trust through proof, not fake scale.
